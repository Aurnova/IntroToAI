*(Instructor: replace the outline below with your actual syllabus.)*

## Course overview

**Intro to AI** introduces core ideas and practice in artificial intelligence. Students use GitHub Codespaces for a uniform environment and optional AI assistants (coding + tutor) powered by OpenRouter.

---

## Learning objectives

By the end of the course, students will be able to:

- Explain basic AI/ML concepts and terminology.
- Use a modern dev environment (Codespaces) and AI tools (Continue, Tutor AI) effectively.
- Complete programming and written assignments that demonstrate understanding.
